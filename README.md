# SPiD API Java client

A Java client for the SPiD API.

## Usage

To use the spid-client-java in your projects include it in your pom.xml.

```
<dependency>
    <groupId>no.spid</groupId>
    <artifactId>no.spid.api.client</artifactId>
    <version>1.2</version>
</dependency>
```
